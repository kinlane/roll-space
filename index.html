Roll Dice as a Service

Have you ever needed to roll dice and have no dice? Do you need to automate the rolling of dice? Do you need some time to kill? I obviously did.

But how do I roll some dice?

Roll one set of dice!

curl http://roll.space/?dice=3d6

Response


{
  "3d6": {
    "sum": 8,
    "rolls": [
      2,
      3,
      3
    ]
  }
}

Roll a lot of dice!

curl http://roll.space/?dice=3d6,2d20,1d100

Response


{
  "3d6": {
    "sum": 8,
    "rolls": [
      2,
      3,
      3
    ]
  },
  "2d20": {
    "sum": "31",
    "rolls": [
      20,
      11
    ]
  },
  "1d100": {
    "sum": "50",
    "rolls": [
      50
    ]
  }
}
